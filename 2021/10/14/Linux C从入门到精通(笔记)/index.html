内存

内存申请

  1. type *buffer = (type *)malloc(size);，申请长度为size子节的内存块，返回指针形式为type；
  2. type *buffer = (type *)calloc(n, size(type))，动态存储区分配n个长度为size(type)的内存块；
  3. type *buffer = (type *)realloc(*mem_address, newsize)，调整mem_address所指向区域的大小为newsize长度。
  4. (type *)换成(void *)应该会自动转换。
  5. memset(*ptr, char ch, n)，将ptr所指向地址内的所有内容换成ch，n是数组的大小。应该就是重复拷贝或者重复赋值。
  6. free(buffer)
